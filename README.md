# aeronet

[![crates.io](https://img.shields.io/crates/v/aeronet.svg)](https://crates.io/crates/aeronet)
[![docs.rs](https://img.shields.io/docsrs/aeronet)](https://docs.rs/aeronet)

A *light-as-air* networking library with first-class support for Bevy, enabling abstracting over
different transport methods while maintaining a simple API surface.
